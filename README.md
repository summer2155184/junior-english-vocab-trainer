# 初中英语单词暑假学习系统 V1

这是一个纯静态、完全离线的英语单词练习网站。没有账号、服务器、数据库或联网请求。学习进度、错词和家长修正均保存在当前浏览器的 `localStorage` 中。

## 文件

- `index.html`：网页入口
- `style.css`：页面样式
- `app.js`：学习、测试、错词和统计逻辑
- `vocab_data.js`：由 `vocab_master.csv` 筛选生成的离线词库

当前构建使用教材顺序中的可测试词条；词组、疑似 OCR 残留和没有中文释义的项目不会进入测试。

每道题会同时显示中文释义和中文词性，例如“名词”“动词”“形容词”。多词性项目会并列显示。少量教材未标注词性的独立单词（如 `hello`、`hi`）已按常规用法补充词性；无词性且呈词组形态的项目不进入考试。

## 如何启动

最简单的方法：双击 `index.html`，使用浏览器打开。

如果浏览器对本地文件有限制，可以在该目录启动任意静态文件服务器。例如电脑已安装 Python 时：

```text
python -m http.server 8000
```

然后打开：

```text
http://localhost:8000
```

网站运行时不访问互联网。

## 学习模式

- 轻松模式：10 个新词，最多 5 个复习词，最多 5 个错词
- 标准模式：15 个新词，最多 5 个复习词，最多 5 个错词
- 冲刺模式：20 个新词，最多 5 个复习词，最多 5 个错词

Day 按完成次数推进。每个 Day 的新词从上次结束位置继续，因此中途切换学习模式不会重复或跳过新词。前 7 个已完成 Day 的新词用于随机复习。

遇到完全不会的题，可以点击“不会，跳过”。系统会显示正确答案、按答错计入错词库，并自动进入下一题。

连续答对时会按次数显示不同鼓励，例如“你好棒，真厉害”“3 连击了”“你好 6 呀”“7 连击，完胜”“10 连击，满分气势”。超过 10 题后会继续按实际次数显示连击提示。答错或跳过时会显示“没关系，再接再厉”等提示，并重置连续答对计数。

首页提供“语音鼓励”按钮，默认关闭。开启后，网站使用浏览器内置的中文语音朗读鼓励文字，不联网，也不下载音频。开关状态保存在 `localStorage`。如果浏览器不支持语音合成，按钮会显示“不支持”，文字反馈仍可正常使用。

刚开始使用或错词不足时，复习题、错词题可能少于 5 道，首页会显示实际题量。

## 如何修改词库

日常发现 OCR 错误时，在测试题右上角点击“修正”，修改英文和中文。修正只保存在当前浏览器，并会同步到错词库和后续测试。

如需永久修改基础词库：

1. 修改源文件 `D:\czyy\ch_result\02_clean\vocab_master.csv`。
2. 重新生成 `vocab_data.js`。
3. 替换网站目录中的旧 `vocab_data.js`。

也可以直接编辑 `vocab_data.js` 中对应项目的 `word` 和 `meaning`，但应保持 JavaScript/JSON 格式有效。

## 如何导出错词

1. 首页点击“查看错词库”。
2. 点击“导出错词 CSV”。
3. 浏览器会下载包含英文、中文、错误次数、连续答对次数和最近错误时间的 CSV 文件。

## 如何清空错词库

首页进入“查看错词库”，点击“清空错词库”。系统会连续确认两次，确认后只清空错词，不影响 Day 进度和家长修正。

## 如何重置学习进度

首页点击“重置学习进度”，连续确认两次。系统会重置 Day、学习统计、当前任务和错词库，但保留家长已做的 OCR 修正及当前学习模式。

若要连家长修正也全部删除，可打开浏览器开发者工具，删除本站点 `localStorage` 中的 `vocabSummerSystemV1`，或清除该网页的站点数据。

## 数据迁移提醒

`localStorage` 按打开方式和域名隔离。直接双击打开与通过 `http://localhost:8000` 打开，可能被浏览器视为两个不同存储空间。长期使用时请固定一种启动方式，并定期导出错词 CSV。
